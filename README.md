# germany_study_advisor
为有意赴德留学的国际学生，尤其是讲中文的申请者，提供留学咨询。  

Advises prospective international students, especially Chinese-speaking applicants, on studying in Germany.  

本工具仅作为咨询辅助，并非正式的录取决定。德国大学的规章制度、学费政策、APS要求、冻结账户金额、截止日期及语言要求均可能发生变动。如需获取最新或关键信息，建议通过大学官网、uni-assist、anabin/KMK、APS、DAAD、德国驻外使领馆及学生服务处（Studentenwerk）等官方渠道核实信息。  

This skill is an advising aid, not an official admissions decision. German university rules, tuition policies, APS requirements, blocked-account amounts, deadlines, and language requirements can change. For current or high-stakes facts, the skill is designed to verify information from official sources such as university pages, uni-assist, anabin/KMK, APS, DAAD, German missions, and Studentenwerk pages.
## 安装方法
使用codex输入：安装 https://github.com/JulioFang/germany_study_advisor 的skill

## 使用方法

安装完成后，在 Codex 中这样调用：

```text
使用 $germany-study-advisor 帮我检查德国留学申请条件。
```

为了获得更准确的判断，建议尽量提供以下信息：

- 申请阶段：本科、硕士、Studienkolleg、博士、交换或语言/预科项目
- 目标专业或项目名称
- 目标大学或项目官网链接
- 你的教育背景国家/地区
- 当前或已完成学历
- 本科/高中专业、成绩单、GPA 或均分
- 成绩换算规则，例如 `85/100`、`3.4/4.0`、`2.1/5.0`
- 语言成绩，例如 IELTS、TOFL、TestDaF、DSH、Goethe、telc
- APS 状态，如果适用
- 预算和偏好的城市/州
- 目标入学季，例如 `2026 冬季学期` 或 `2027 夏季学期`

## 示例提问

### 检查入学资格

```text
使用 $germany-study-advisor 帮我判断我是否符合这个德国硕士项目的申请条件。

项目：某大学 MSc Computer Science
项目链接：[粘贴官网链接]
我的背景：中国某大学软件工程本科
成绩：85/100
语言：IELTS 7.0
APS：还没有完成
目标入学季：2027 冬季学期
```

### 分数换算

```text
使用 $germany-study-advisor 帮我把我的成绩换算成德国分数。

我的成绩是 85/100。
满分是 100。
及格线是 60。
分数越高越好。
```

### 项目匹配

```text
使用 $germany-study-advisor 帮我推荐适合我的德国硕士项目。

背景：机械工程本科
GPA：3.3/4.0
语言：IELTS 6.5，暂时没有德语成绩
预算：优先考虑公立大学
偏好：应用型项目，希望有较好的工业界联系
目标入学季：2027 冬季学期
```

### 费用检查

```text
使用 $germany-study-advisor 帮我估算这个德国大学项目的费用。

大学：[大学名称]
项目：[项目名称]
城市：[城市]
申请人身份：非欧盟国际学生
```

## 再次重要提醒

这个 skill 提供的是申请规划和条件核查辅助，不等同于大学、uni-assist、APS 或使领馆的官方决定。

德国大学的入学要求、学费政策、APS 要求、保证金金额、申请截止日期和语言要求都可能变化。涉及具体项目或重要申请决策时，请以大学官网、uni-assist、anabin/KMK、APS、DAAD、德国使领馆或 Studentenwerk 等官方来源为准。
